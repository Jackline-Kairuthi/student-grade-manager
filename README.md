# Student Grade Manager

A beginner‑friendly Python console application that allows you to:

- Add students and their marks (English, Math, Science)
- Calculate total marks and assign grades
- Compute individual student averages
- Display class and subject averages
- Search for a student by name
- View all stored student data

This project was built to practice:
- Loops (`for`, `while`)
- Functions
- Lists and dictionaries
- Conditional statements
- Basic data analysis logic
